# Astraea Libra

> 这是 GinShio 毕业设计项目 -- Astraea OJ 的判题机部分，目前状态为 **屎山** 且不可用状态。当下没有时间维护，在未来的某个时间继续开发。

利用 C++ 与 Linux 提供的 Namespaces、CGroups、OverlayFS 实现的 Astraea 判题服务。
